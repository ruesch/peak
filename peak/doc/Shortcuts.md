# Peak Shortcuts

## 1. Text Editing

- Ctrl-C            Copy selection to clipboard.
- Ctrl-X            Cut selection to clipboard.
- Ctrl-V            Paste from clipboard.
- Ctrl-Z            Undo last change.
- Ctrl-Y            Redo last undone change.
- Ctrl-Shift-Z      Redo last undone change.
- Ctrl-A            Select all.
- Ctrl-U            Delete the entire current line.
- Ctrl-W            Delete the word before the cursor.
- Backspace         Delete character before cursor.
- Delete            Delete character at cursor.
- Tab               Insert a tab character.
- Enter             Insert a newline.
- Esc               Cancel selection, or reselect newly typed text.


## 2. Navigation

- Arrow             Move cursor up, down, left, right.
- Ctrl-Left/Right   Move cursor by word.
- Home              Move to the beginning of the line.
- End               Move to the end of the line.
- PgUp / PgDn       Scroll and move cursor by one screen.


## 3. Search

- Ctrl-F            Run Look on the current selection.


## 4. Mouse

Three mouse buttons:

- Button 1 (Left)   Select text. Drag handle or scrollbar.
- Button 2 (Right)  Plumb: open path, URL, or search for word.
- Button 3 (Middle) Execute: run word or selection as a command.

Scrollbar (left edge of window body):

- Left              Scroll up.
- Right             Scroll down.
- Middle            Jump to the corresponding position in the file.

Handle (left edge of each window):

- Left (no drag)    Grow window by half its height.
- Right (no drag)   Maximize window, hiding others below.
- Middle (no drag)  Expand window, keeping others at minimum size.
- Drag              Move or resize the window.


## 5. Terminal Windows

In terminal windows, most keystrokes go to the shell. Hold Alt to use
Peak shortcuts:

- Alt+Ctrl-C        Copy selection to clipboard.
- Alt+Ctrl-V        Paste from clipboard.
- Alt+Ctrl-F        Run Look on the current selection.
- Alt+PgUp          Scroll up.
- Alt+PgDn          Scroll down.

Mouse shortcuts work with Ctrl held:

- Ctrl+Right-click  Plumb.
- Ctrl+Middle-click Execute.
